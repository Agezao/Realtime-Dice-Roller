## Getting started

### Install Global Dependancies
  * [Node.js](http://nodejs.org)

### Running the project
  * [clone the repo](git clone https://github.com/Agezao/Realtime-Dice-Roller.git)
  * cd to project folder
  * run `[sudo] npm install` (first time users)
  * run `[sudo] npm start`
